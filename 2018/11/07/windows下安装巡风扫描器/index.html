一款开源的企业内网的漏洞快速应急、巡航扫描系统。

前言

前几天看有人在群里发了个图片，像这样的：

得知是一款叫巡风的开源web扫描器，随即到GitHub上一搜，找到了

项目地址：
https://github.com/ysrc/xunfeng

Windows 安装指南:
https://github.com/ysrc/xunfeng/blob/master/docs/install/Windows.md

之前安装过MongoDB，默认端口是27017，但是这里设定的端口是65521，所以安装了好几遍没安装好，百度也没有找到好的教程，自己对MongoDB也不熟悉，所以又去学习了一下MongoDB，然后自己折腾了两天才把它安装起来，故写篇博客记录一下。

安装

将下载下来的zip解压到D盘并命名为xunfeng

首先需要有python2环境，记得添加环境变量，如：D:\python27 和 D:\python27\Scripts

cd D:\xunfeng

使用pip安装 python 依赖库, 这里使用了豆瓣的 pypi 源

另外，需要有MongoDB数据库，我把MongoDB安装在D盘（D:\MongoDB）
然后在其安装目录的data目录下建立了一个db目录。
具体如何安装及使用，可以参考我的另一篇文章。
