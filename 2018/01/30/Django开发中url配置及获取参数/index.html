Django开发中url配置及获取参数

2018-01-30
python django web
  1. 项目开发中配置 URL
  2. 关于获取 URL 参数

项目开发中配置 URL

  • 1、在主的 urls.py 文件中配置 include 引入组件(App)组件内的 urls.py

  • 2、在各个组件(App)中定义视图

  • 3、关于配置URL正则的规则

    • 1、主 URL 开始地方不要加/
    • 2、主 URL 后面地方要加/
    • 3、组件(App)的 URL 前面不要加/
    • 4、主 URL 后面不要加$
    • 5、组件(App)后面要加$

关于获取 URL 参数

由于Django中 URL 中不分get还是post请求的，所有没有query参数，全是path参数

  • 1、在urls.py中写上使用关键字参数

  • 2、在视图函数中接收参数

  • 3、一般会在视图函数中默认一个值

  • 4、多个参数的urls.py情况

  • 5、多个参数的视图类

  • 6、页面跳转中配置参数参考模板

在 urls.py 中定义 name 的作用

    2、在页面中做页面跳转参考模板
