SublimeText3 快捷键

2018-01-18
编辑器 SublimeText3
  1. 编辑
    1. 插入行
    2. 转换
    3. 删除
    4. 撤销与恢复
    5. 缩进
    6. 注释
  2. 选择
  3. 查找和替换
  4. 跳转
  5. 书签
  6. 窗口
  7. 屏幕
  8. 文件
  9. 其他操作
    1. 格式化粘贴

好记性不如烂笔头。自己总结写一遍。

编辑

Ctrl + 鼠标左键 多个光标同时编辑

Shift + 鼠标右键拖动 分割选择列

Ctrl + Alt + ↑/↓ 同时编辑多行

Ctrl + Shift + ↑/↓ 将光标所在的行上/下移

Ctr + Shift + D 复制光标所在行到下一行（也可以将选中的内容复制到选中的内容后面）

Ctrl + Shift + ; 清除光标相关的父标签

Alt + . 补一个关闭标签，如补个</div>

Ctrl + J 将选中的行合并成一行（或将下一行合并到本行）

插入行

Ctrl + Enter 在下方插入一行
Shift + Enter 将光标后面的内容移动到下一行（在上方插入一行）
Ctrl + Shift + Enter 在当前行上面增加一行并跳至该行

转换

Ctrl + T 左右字母互换
Ctrl + K + U 转化为大写（自动识别单词）
Ctrl + K + L 转化为小写（自动识别单词）

删除

Ctrl + Shift + k 删除光标所在的行
Ctrl+X 能剪切当前行，也能当作删除当前行
（当选择内容时，为剪切功能。多数编辑器是使用Ctrl+d 来删除光标所在的行）

Ctrl + K Backspace 从光标处删除至行首
Ctr l+ KK 删除光标后面的内容至行尾

撤销与恢复

Ctrl + Z 撤销
Ctrl + Y 恢复撤销（有些编辑器是用Ctrl+R还原）

缩进

Ctrl + [ 将光标所在的行或选中的内容向左缩进（等同于Tab ）

Ctrl + ] 将光标所在的行或选中的内容向右缩进（等同于Shift + Tab）

注释

Ctrl + / 用//注释当前行（或取消注释）
Ctrl + Shift + / 当前位置插入用/**/注释（或者自动识别多行并将其注释）

选择

Ctrl + a 全选

按住鼠标中间的滚轮拖动，可以垂直选择

Shift+ ←/→ 逐个字符选择
Ctrl + Shift + ←/→ 进行逐词选择
（Ctrl + ←/→ 进行逐词移动光标）

Ctrl + D 选中光标周围的一个单词。
选中一个关键字后按Ctrl + D，多次按D会依次选择下一个和当前选中的单词。
按F3跳到下一个相同单词的位置，Shift + F3进行回退到上个位置。
可以配合 Alt + F3一次选择和当前选中的所有一样的单词。

Ctrl + L：选择光标所在的行，重复可依次增加选择下一行

Ctrl+Shift+M 选中当前括号里的内容，重复可选着括号本身

在html中，Ctrl + Shift + a 选中光标所在位置父级标签里的内容
例如：光标在下面的这一行的任何位置都可以选中Document
\Document\
Alt + Shift + w 将选中的内容容标签包裹(默认使用的是p标签)

查找和替换

（如果有窗口弹出都是Esc退出弹出窗口）

搜索的时候各个关键字可以用”|”隔离开，这样可以一下子匹配很多关键字。

Ctrl + F 调出搜索框

Ctrl + H 调出替换框进行替换

Ctrl + Shift + H 输入替换内容后，替换当前关键字

Ctrl + Alt + Enter 输入替换内容后，替换所有匹配关键字。(注意此时如果鼠标焦点在编辑窗口中，则替换失败，将鼠标焦点调到替换框中，Ctrl + Alt + Enter才会起作用)

在搜索框输入关键字后Enter跳至关键字当前光标的下一个位置，Shift + Enter跳至上一个位置，Alt + Enter选中其出现的所有位置（同样的，接下来可以进行快速替换）。
Sublime Text的查找有不同的模式：Alt + C切换大小写敏感（Case-sensitive）模式，Alt + W切换整字匹配（Whole matching）模式，除此之外Sublime Text还支持在选中范围内搜索（Search in selection），这个功能没有对应的快捷键，但可以通过以下配置项自动开启。
"auto_find_in_selection": true

跳转

Ctrl + P 列出当前打开的文件（或者是当前文件夹的文件），
输入文件名或按上下键盘选择，然后按Enter跳转至该文件，
输入@symbol跳转到symbol符号所在的位置，
输入#keyword跳转到keyword所在的位置，
输入:n跳转到文件的第n行

Ctrl + R 列出当前文件中的符号（例如类名和函数名，但无法深入到变量名），输入符号名称Enter 即可以跳转到该处。会列出Markdown文件的大纲

F12 快速跳转到当前光标所在符号的定义处（Jump to Definition）。比如当前光标所在为一个函数调用，F12 会跳转至该函数的定义处。

Ctrl + G 输入行号以跳转到指定行

Ctrl+M 跳转到括号另一半。

Alt + - 回到上次编辑位置

Ctrl + ←/→ 进行逐词移动光标
（Ctrl + Shift + ←/→ 进行逐词选择）

书签

Ctrl+F2 设置书签
F2 下一个书签
Shift + F2 上一个标签
Ctrl+Shift+F2 清除全部书签

窗口

Ctrl + N 在当前窗口创建一个新标签

Ctrl + Shift + N 创建一个新窗口（可能和搜狗输入法快捷键冲突）

Ctrl + W 关闭标签页，如果没有标签页了，则关闭该窗口

Ctrl+Shift+W 关闭所有打开文件

Ctrl + Shift + T 恢复刚刚关闭的标签。

Ctrl + PageDown 向左切换当前窗口的标签页。
Ctrl + PageUp 向右切换当前窗口的标签页

屏幕

F11 切换普通全屏

Shift + F11 切换无干扰全屏

Alt+Shift+数字键 分屏
（1，2，3，4为竖屏，5是上下各两个屏共4屏，8为上下两横屏，9为三个横屏）

Ctrl + Shift + 数字键： 将当前屏移动到指定屏

Ctrl +数字键： 跳转到指定屏

文件

Ctrl+n 新建文件

Ctrl + O 打开文件

Ctrl + P 列出当前打开的文件（或者是当前文件夹的文件），
输入文件名或按上下键盘选择，然后按Enter跳转至该文件，
输入@symbol跳转到symbol符号所在的位置，
输入#keyword跳转到keyword所在的位置，
输入:n跳转到文件的第n行

在左侧显示目录
点击菜单栏“view”——“Side Bar”——“shaw side bar”
之后可以把文件夹拖到左侧的目录栏中
修改侧边栏颜色
默认的左侧的目录栏是白色，不是很护眼模式啊。
Ctrl+Shift+P -> Install Package
用Package Control安装SyncedSidebarBg插件，安装成功后会自动同步背景颜色。

其他操作

Ctrl + ` 打开 Sublime Text 控制台（Esc退出）

Ctrl + Shift + P 打开命令面板（Esc退出）

Alt 光标调到菜单栏，↑↓←→移动光标

Ctrl + 加号/减号 临时调整字体大小

F6 检测语法或单词错误

格式化粘贴

通过以下三步就可以让粘贴的内容保持正确的缩进

  1. 复制原文
  2. 将光标定位到要粘贴的起始位置
  3. Ctrl+Shift+V
