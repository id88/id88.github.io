  • 在Windows操作系统中 \r\n 表示换行。

r 是读取文本文件，并且是 ASCII 编码的文本文件
要读取非 ASCII 编码的文本文件，就必须以二进制模式打开，参数为 rb

r：读

w：写

a：追加

r+ 等价于 r+w（可读可写，文件若不存在就报错(IOError)）

w+ 等价于 w+r（可读可写，文件若不存在就创建）

a+ 等价于 a+r（可追加可写，文件若不存在就创建）

对应的，如果是二进制文件，就都加一个 b 就好啦：

rb　　wb　　ab　　rb+　　wb+　　ab+
